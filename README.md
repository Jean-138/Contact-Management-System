# Contact Management System

CLI-based contact management system developed in Python.

## Description

This project is a contact management system created in Python for terminal use. It allows users to add, edit, delete, display and export contacts in a simple and organized way.

The system was developed during Python studies and inspired by academic activities from FIAP (Faculdade de Informática e Administração Paulista), with a focus on dictionaries, lists, functions, loops, conditions and file handling.

## Features

* Add new contacts
* Add multiple contact methods
* Edit contact names
* Edit phone numbers, emails or addresses
* Display a specific contact
* Display the full contact list
* Delete contacts
* Export contacts to TXT
* Export contacts to JSON
* Import contacts from JSON

## Technologies Used

* Python
* JSON
* File handling
* Dictionaries and lists

## How to Run

1. Clone this repository
2. Open the project folder in VS Code
3. Run the following command in the terminal:

```bash
python main.py
```

## Project Structure

* `main.py` → Main system file
* `contacts.json` → Example exported contact file
* `contacts.txt` → Example exported text file
